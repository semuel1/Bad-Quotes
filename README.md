# Bad-Quotes
Breaking Bad characters and their "bad" quotes 

# MVP Goals
1. Create a homepage that allows you to make a profile
2. As a user you will be able to save your favorite characters and their quotes.
3. You will be able to search by characters or qoutes, via a dropdown menu
4. Upon clicking on a character you will gain information about that character(Nickname and life or death status)
5. You will also be able to leave comments for any character!

# Stretch Goals
* Slide in menu for profile creation.
* Sound of each quote.
* Allow replies in character comment section.
* 

# User Stories
* As a user, I want to create an account
* As a user, I want to log in and out
* As a user, I want to see each characters photo
* As a user, I want to browse all Breaking Bad Characters and their quotes
* As a logged in user, I want a favorites list, where I can place my favorite characters and their quotes
